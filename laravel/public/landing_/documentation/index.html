Overview

The landing page’s main goal is to convert visitors into customers, for that reason the template is specially designed and has got many features designed specially for the landing page. The template is highly and easily customizable, so it will be perfect for your product, service or company.

Landing Page is divided into sections:

  • 1- Simple Header with Logo and Phone Number. Without menu, this way the user is not distracted and immediately observes the Call To Action to Subscribe or Purchase the service or product.
  • 2- Title Call to Action.
  • 3- Slider (or Video).
  • 4- Subscription form with MailChimp.
  • 5- Customers.
  • 6- Promotions & Discounts.
  • 7- Slider Testimonials with photo.
  • 8- Features & FAQs.
  • 9- Screenshots of the Services or Products.
  • 10- Call To Action with Purchase button.
  • 11- Footer with 3 sections (About Us, Company, Contact Us) and Social Icons.

eLite has a responsive layout that looks great on mobile and tablet devices. Enjoy!

CSS Files and Structure

The whole structure is based on Bootstrap. We using three custom CSS files in this theme:

wide.css (or boxed.css)

Controls the disposition of the elements in the structure. It is the main style sheet and its use is mandatory. For example, if you want the page to see the width of the screen choose wide.css, but if you want to display with wide margins alongside choose boxed.css

[color].css

This style sheet allows you to choose a variety of colors for both fonts for background and other elements. You can choose the color scheme you want to invoking any css of the folder assets/css/colors. For example: blue.css

responsive.css

This file contains some features added to certain page elements display correctly on mobile devices. Its use is mandatory.

Other css files

The other css files are used to control the appearance of various components such as slider, web fonts, and support for Internet Explorer.

MailChimp setting

  1. Open the file php/mcapi-subscribe/inc/store-address.php
  2. Go to the line 24 and replace "your_apikey" with an API Key retrieved from here: http://admin.mailchimp.com/account/api/.
  3. Go to the line 27 and replace "my_list_unique_id" with the List Unique Id for the Mailing List. For obtain the id please following this steps:
    • Go to http://admin.mailchimp.com/lists/.
    • Click the "settings" link for the list.
    • The Unique Id is at the bottom of that page.
  4. Save the file and try the suscription form.

Important: This feature requires an active account in service MailChimp.
